Projects

Summarize YouTube video with LLM

This project aims to develop an AI-powered tool that automatically generates concise and informative summaries of YouTube videos using Large Language Models (LLMs). The system extracts audio from videos, converts speech to text, processes the transcript, and generates a structured summary.

Detecting DeepFake Videos and Images

This project focuses on detecting DeepFake videos and images using deep learning techniques. DeepFakes, created using generative models like GANs and diffusion models, pose serious threats to privacy, security, and misinformation. This system leverages advanced computer vision models to identify manipulated media with high accuracy.

Counterfeit Product Detection Using Algorand Blockchain

This project utilizes the Algorand blockchain to develop a secure and transparent system for detecting counterfeit products. By leveraging the immutability and security features of blockchain technology, this system enables consumers and businesses to verify the authenticity of products, reducing the risk of counterfeiting and promoting trust in the supply chain.
